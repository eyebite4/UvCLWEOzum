# Python图书馆座位预约系统简介

## 核心功能
- 登录注册
- 座位预约
- 预约记录
- 警告记录
- 签到
- 自习室选择
- 用户管理
- 预约管理
- 警告管理
- 自习室管理
- 提示管理

## 适用场景
- 课程设计
- 大作业
- 图书馆自习室座位预约

## 技术特点
- 使用Django框架进行开发
- 基于MySQL数据库进行数据存储
- 采用HTML、CSS、JavaScript进行前端页面展示和交互
